# Мессенджер

Современный мессенджер с поддержкой личных и групповых чатов, построенный на FastAPI и Flet.

## Возможности

### 🔐 Аутентификация
- Регистрация и вход пользователей
- JWT токены для безопасности
- Управление профилем

### 💬 Типы чатов
- **Личные чаты** - приватное общение между двумя пользователями
- **Групповые чаты** - закрытые группы с приглашениями
- **Публичные комнаты** - открытые каналы для всех

### ⚡ Реальное время
- WebSocket соединения для мгновенных сообщений
- Уведомления о приглашениях
- Автоматическое переподключение

### 🎨 Современный UI
- Интуитивный интерфейс с вкладками
- Адаптивный дизайн
- Темная/светлая тема

## Быстрый запуск

### Сервер
```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### Клиент
```bash
python -m client.main
# или
flet run client
```

### Миграции базы данных
```bash
python migrate.py
# или
alembic upgrade head
```

## Архитектура

### Бекенд (FastAPI)
- **Модели**: User, Room, RoomMember, Message
- **Типы комнат**: personal, group, public
- **API**: REST endpoints + WebSocket
- **База данных**: SQLAlchemy + Alembic

### Фронтенд (Flet)
- **Вкладки**: Личные, Группы, Публичные
- **Реал-тайм**: WebSocket клиент
- **Состояние**: Централизованное управление

## Разрешения

### Личные чаты
- Автоматическое создание при первом сообщении
- Только 2 участника
- Нельзя приглашать других

### Групповые чаты
- Создает владелец
- Приглашения по разрешению
- Настройки доступа

### Публичные комнаты
- Может присоединиться любой
- Видны в общем списке
- Модерация владельцем
